Project Warm Up

1. Open the Chrome DevTools Console

Type the keyboard shortcut Ctrl Shift i if you're on Windows, or Command Option i if you're on a Mac.

2. Complete the steps in the comments in the script.js file

Just take it one step at a time. Write your code, save the file, and check your results.

3. Click the "Test" button to see results

Clicking the "Test" button will run your code since you're writing it inside of an event listener that's been provided for you.

4. If you run into trouble or have questions, be sure to reach out in Slack
